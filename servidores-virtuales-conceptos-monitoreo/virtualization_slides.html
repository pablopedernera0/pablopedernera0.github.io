¿Qué es la Virtualización?

  • Definición: Tecnología que permite ejecutar múltiples sistemas operativos en un solo hardware físico
  • Beneficios: Mejor utilización de recursos, reducción de costos, mayor flexibilidad
  • Abstracción: Capa que separa el software del hardware físico
  • Consolidación: Múltiples cargas de trabajo en menos servidores físicos
VMware - Conceptos de Virtualización Microsoft Docs - Virtualización

Hipervisores: El Corazón de la Virtualización

  • Hipervisor: Software que crea y gestiona máquinas virtuales
  • Función principal: Asignar recursos físicos a VMs
  • Aislamiento: Garantiza que las VMs no interfieran entre sí
  • Gestión de recursos: CPU, memoria, almacenamiento y red
Red Hat - ¿Qué es un Hipervisor?

Tipos de Hipervisores

🏢 Tipo 1 (Bare Metal)

  • Se ejecuta directamente sobre hardware
  • Mayor rendimiento
  • Menor latencia
  • Uso en centros de datos
  • Ejemplos: VMware ESXi, Hyper-V Server, Citrix XenServer

💻 Tipo 2 (Hosted)

  • Se ejecuta sobre un OS host
  • Más fácil de instalar
  • Mejor para desarrollo/testing
  • Menor rendimiento
  • Ejemplos: VMware Workstation, VirtualBox, Parallels

¿Por qué Monitorear?

  • Prevención: Detectar problemas antes de que afecten usuarios
  • Rendimiento: Optimizar recursos y identificar cuellos de botella
  • Disponibilidad: Garantizar SLAs y tiempo de actividad

Estrategias de Respaldo

  • Regla 3-2-1: 3 copias, 2 medios diferentes, 1 offsite
  • Full Backup: Copia completa de todos los datos
  • Incremental: Solo cambios desde último backup
  • Differential: Cambios desde último full backup
  • Snapshots: Punto en el tiempo para recovery rápido

Snapshots vs Backups

📸 Snapshots

  • Propósito: Recovery rápido
  • Ubicación: Mismo storage
  • Velocidad: Instantáneo
  • Retención: Corto plazo (días)
  • Limitación: No protege contra fallos de storage

💾 Backups

  • Propósito: Protección de datos
  • Ubicación: Storage separado
  • Velocidad: Proceso más lento
  • Retención: Largo plazo (meses/años)
  • Ventaja: Protección completa
